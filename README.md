# Wikidot-Archive

**关于类型的解释：** <br>
html - 网页的预览文件<br>
source - wikidot语法源代码的txt文件<br>
file - 网站页面中存储的文件<br>
all - 以上三种类型都备份

除all分类外，其他分类在同一zip中的命名可以用连接线连接。

例：<br>
TG的次元空间_backup_2023-01-30_html<br>
TG的次元空间_backup_2023-02-25_source-file<br>
TG的次元空间_backup_2023-03-04_all
